## 🌟 Featured Project

### 📦 [PCDrop](https://github.com/udaysai12/PCDrop) — Local Wi-Fi File Transfer Tool
Cross-platform, open source file transfer over local Wi-Fi — no internet, no cloud, no size limits.

- ⚡ Chunked **parallel transfers** for high throughput
- 📡 **mDNS/UDP** based automatic device discovery
- 🔒 **SHA-256** integrity verification on every file
- 🖥️ Desktop app: **Electron + React + TypeScript**
- 📱 Mobile app: **Flutter**
- 🔥 Windows **hotspot fallback** for networks without shared Wi-Fi
